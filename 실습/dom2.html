Display

Block

tag - h1~h6, div, header, main, footer, nav, ul, li, dl, dt, dd, figure, figcaption

특징 - 스타일 제약이 없음, 한줄 내리기 효과 있음

사용 - 그룹 또는 의미에 따라 달리 사용함

inline

tag - img,a,span,em,strong,b,code

특징 width,height,margin,padding 스타일 제약 있음 (4개 중에 한가지라도 사용해야 한다면 inline-block으로 변경)

사용 - 가로형 콘텐츠일때 사용됨, block요소의 상위 요소로 위치할 수 없음

inline-Block

tag - input,select,button,textarea

사용 - 회원정보입력,버튼 등의 입력값에 사용

특징 - 가로형 콘텐츠를 만들기 위해 block 또는 inline요소들에게 사용할 수 있음.
줄내림 효과가 없을뿐 block요소와 유사함(스타일 제약이 없음)
